## Context

- Current status: !`git status`
- Current diff: !`git diff HEAD`
- Current branch: !`git branch --show-current`

## Critical Rules

1. Create a meaningful commit message based on the changes above.
2. Ensure it follows the [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/#summary) specification.
3. Do not mention that it was generated by an AI.
4. Ensure to sign the commit with `-s` so it is signed with my information.
5. If the issue-url ($1) is provided, add `Relates to $1` as the first line after the commit title to associate the commit to its issue.
6. **NEVER git commit or git push without explicit user approval** - ALWAYS ask first
