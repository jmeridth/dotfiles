# My dotfiles

Used with my local dev environment.

Uses [chezmoi](https://www.chezmoi.io/)

## Local Setup

To Install:

`sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply $GITHUB_USERNAME`

includes the following

- installing chezmoi
- sets up aliases
- commands and apps via Homebrew
- gitconfig (specific to me)
- and others...
